# Install
`npm install`

# Run
  - tslint check whole project
    - `npm run lint`
  - tsc build
    - `npm run build`

# Test
  - test
    - `npm run test`
  - coverage
    - `npm run coverage`
  - test with real instance of postgrest
    - `docker-compose up`
    - `npm run test:all`
